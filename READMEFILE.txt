QR Code Generator using React

Description:

	This project is a QR code generator built using React. It allows users to create QR codes for various purposes such as sharing URLs, text, contact information, and more.

Features:

1.Generate QR codes for URLs, text, and other data formats.
2.Customize QR code size, and correction level.
3.Download generated QR codes as images for offline use.


Usage:

1.Start the development server:
	npm start
2.Open your browser and navigate to http://localhost:3000.
3.Select the type of data you want to encode into the QR code (e.g., URL, text, contact).
4.Enter the required data (e.g., URL, text, contact information) in the input field.
5.Customize the QR code size, color, and error correction level if desired.
6.Click on the "Generate QR Code" button to create the QR code.
7.Download the generated QR code by clicking on the "Download" button.
